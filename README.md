# Party
Makes your terminal party. 

![funny img](http://i.giphy.com/gpivFuK6iE3hS.gif)

## Usage

`gem install consoleparty`

For help run: 
`$ party -h`

Simple party:
`$ party`

Silent party:
`$ party --silent`

Party with farewell and custom MC sequence
`$ party -farewell "bye bye" "yo" "let's go" "yo"`
